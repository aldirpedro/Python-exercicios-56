Exercícios de Programação em Python

1. Contar os Caracteres em uma String

Escreva um código Python que conte a quantidade de caracteres em uma string.

# Exemplo de código em Python:
string = "Exemplo de string"
print("A string tem", len(string), "caracteres.")

Resposta: A string tem 18 caracteres.

2. Converter String para Maiúsculas

Escreva um código Python que converta uma string para maiúsculas.

# Exemplo de código em Python:
string = "Exemplo de string"
print(string.upper())

Resposta: EXEMPLO DE STRING

3. Converter String para Minúsculas

Escreva um código Python que converta uma string para minúsculas.

# Exemplo de código em Python:
string = "Exemplo de string"
print(string.lower())

Resposta: exemplo de string

4. Contar Ocorrências de um Caracter

Escreva um código Python que conte quantas vezes um caractere específico aparece em uma string.

# Exemplo de código em Python:
string = "Exemplo de string"
caracter = 'e'
print("O caractere", caracter, "aparece", string.count(caracter), "vezes.")

Resposta: O caractere e aparece 2 vezes.

5. Verificar se uma Palavra está na Frase

Escreva um código Python que verifique se uma palavra está presente em uma frase.

# Exemplo de código em Python:
frase = "Exemplo de string"
palavra = "de"
if palavra in frase:
    print(f'A palavra "{palavra}" está na frase.')
else:
    print(f'A palavra "{palavra}" não está na frase.')

Resposta: A palavra "de" está na frase.

6. Substituir Palavras em uma Frase

Escreva um código Python que substitua uma palavra por outra em uma frase.

# Exemplo de código em Python:
frase = "Exemplo de string"
nova_frase = frase.replace("string", "texto")
print(nova_frase)

Resposta: Exemplo de texto

8. Dividir uma Frase em Palavras

Escreva um código Python que divida uma frase em palavras.

# Exemplo de código em Python:

9. Capitalizar as Palavras de uma Frase

Escreva um código Python que capitalize a primeira letra de cada palavra de uma frase.

# Exemplo de código em Python:
